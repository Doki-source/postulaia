# PostulaIA

Aplicación web que compara un perfil junior con una oferta laboral y genera un diagnóstico mediante Google Gemini AI.

## Flujo

Formulario web → webhook de Make → Gemini 2.5 Flash-Lite → respuesta JSON → visualización.

## Uso

Abrir `index.html` con un servidor local o desplegar la carpeta en GitHub Pages.

## Tecnologías

- HTML5, CSS3 y JavaScript
- Make
- Google Gemini AI
- Webhook HTTP

## Salida

Puntaje, nivel de compatibilidad, fortalezas, brechas, recomendación, plan de 30 días y resumen para CV.
